🔗 Configuration — Serveur pédagogique
📋 Architecture recommandée (réseau pédagogique fermé) :
1. Placez melec-suivi-v8.html et melec-data.json dans le même dossier partagé sur le serveur
2. Lancez une seule fois le mini-serveur HTTP (voir ci-dessous) sur le poste serveur
3. Chaque enseignant ouvre http://<ip-serveur>:8080/melec-suivi-v8.html dans son navigateur
4. La synchronisation entre postes se fait automatiquement toutes les 10 secondes
▶ Mini-serveur HTTP Python — à lancer UNE FOIS sur le serveur :
# 1. Ouvrir un Invite de commandes dans le dossier du fichier :
cd "\\\\serveur\partage\melec"
# 2. Lancer le serveur (Python est inclus dans Windows depuis Win10) :
python -m http.server 8080 --bind 0.0.0.0
# 3. Accès depuis tous les postes :
http://<ip-du-serveur>:8080/melec-suivi-v8.html
⚠ Remarque : Si Python n'est pas disponible, IIS (gestionnaire de services Internet) peut être activé sur Windows Server via Ajout de rôles → Serveur Web (IIS) et le dossier partagé défini comme répertoire racine.
Le fichier melec-data.json doit être accessible en lecture ET écriture via HTTP (voir Permissions NTFS et requêtes PUT dans IIS).
Terminer par un slash / · Laisser vide si le HTML est déjà ouvert depuis ce serveur (chemin relatif automatique)
